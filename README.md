# NYC Taxi Data Dashboard

An interactive dashboard built using Python, Dash, and Plotly to analyze NYC taxi trip data.
The dashboard provides geospatial visualizations, KPIs, clustering-based hotspots, and data export features.

## Features
- Hour and borough-based filtering
- Pickup hotspot detection using DBSCAN
- Interactive maps and charts
- CSV and PDF download options

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Dash & Plotly

## How to Run
1. Install required libraries
2. Run `python app.py`
3. Open http://127.0.0.1:8050 in browser
